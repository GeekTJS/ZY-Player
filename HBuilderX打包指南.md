# ZY Player 移动端/电视端打包指南

## 项目改造说明

本项目已从 Electron 桌面端改造为兼容 **小米电视** 和 **手机** 的 uni-app 项目。

## 环境准备

### 1. 安装 HBuilderX
- 下载地址：https://www.dcloud.io/hbuilderx.html
- 安装最新版本的 HBuilderX

### 2. 安装 Node.js 依赖
```bash
npm install
```

### 3. 配置 manifest.json
已创建 `manifest.json` 文件，包含：
- 应用名称：ZY Player
- 应用标识：__UNI__ZYPLAYER
- 版本号：2.8.8
- Android 权限配置

## 打包步骤

### 1. 在 HBuilderX 中打开项目
1. 打开 HBuilderX
2. 选择 文件 -> 打开目录
3. 选择本项目根目录 `codeZY-Player`

### 2. 配置应用标识
1. 点击项目根目录下的 `manifest.json`
2. 在 HBuilderX 中可视化配置：
   - 基础配置：应用名称、版本号
   - App图标配置：上传应用图标（建议 1024x1024）
   - App启动图配置：上传启动图
   - App模块配置：根据需要选择模块
   - App权限配置：已预设网络、存储等权限

### 3. 发行打包

#### 打包 Android APK（手机/电视通用）
1. 点击菜单栏：发行 -> 原生App-云打包
2. 选择：
   - Android 包
   - 使用 DCloud 公用证书（测试用）
   - 或使用自有证书（正式发布）
3. 点击"打包"
4. 等待云端打包完成，下载 APK

#### 打包自定义基座（调试）
1. 点击菜单栏：运行 -> 运行到手机或模拟器 -> 制作自定义基座
2. 打包完成后，可以真机调试

### 4. 小米电视安装
1. 将 APK 文件拷贝到 U 盘
2. U 盘插入小米电视
3. 打开小米电视的"高清播放器"或"文件管理"
4. 找到 APK 文件，点击安装
5. 如果提示"禁止安装未知来源应用"：
   - 进入 设置 -> 账号与安全 -> 允许安装未知来源应用

## 电视端适配说明

### 遥控器操作
- **方向键**：上下左右移动焦点
- **确认键**：点击/确认
- **返回键**：返回上一级
- **Home键**：回到首页

### 焦点样式
- 当前焦点元素会有蓝色边框高亮显示
- 焦点会自动滚动到可视区域

## 移动端适配说明

### 触摸操作
- **点击**：选择/播放
- **左右滑动**：切换分类
- **上下滑动**：滚动列表
- **长按**：显示更多选项

## 注意事项

1. **网络权限**：应用需要网络权限获取视频资源
2. **存储权限**：用于缓存视频数据
3. **电视适配**：电视端界面会自动适配大屏幕显示
4. **焦点控制**：电视端使用遥控器方向键控制焦点

## 常见问题

### Q: 打包失败怎么办？
A: 检查 manifest.json 配置是否正确，应用标识是否唯一。

### Q: 电视端无法安装？
A: 确保小米电视允许安装未知来源应用。

### Q: 遥控器无法操作？
A: 确保电视支持标准 Android TV 按键事件。

### Q: 视频无法播放？
A: 检查网络连接，部分视频源可能需要特定解析接口。
